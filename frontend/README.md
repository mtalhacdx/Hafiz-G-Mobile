# Frontend (Foundation)

## Implemented
- React + Vite shell
- Redux store with auth slice
- Axios API client with JWT interceptor
- Protected route flow and Login page
- Required page route structure:
  - Dashboard
  - Products
  - Categories
  - Sales / Billing
  - Purchases
  - Customers
  - Suppliers
  - Expenses
  - Reports
  - Settings
  - Login

## Run
```bash
npm run dev
```
